# CS120B-Embedded-Systems-Labs
Luke McFadden's Repo for CS120B Lab, UCR Fall 2020.

This repo is forked directly from my university github account.

Various componenets are utilized throughout the labs:
- ATMEGA1284-PU
- LCD Displays
- Shift registers
- Potentiometers
- Joysticks
- Resistors
- 4x4 Keypad
- Speakers/buzzers
- LEDs
- Buttons
